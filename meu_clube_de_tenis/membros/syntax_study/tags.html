Tags de Template

Em templates do Django, você pode executar lógica de programação como if statements e for loops.

Essas palavras-chave, if e for, são chamadas de "tags de template" no Django.

Para executar tags de template, as envolvemos em colchetes {% %}.

Exemplo: if statement

templates/template.html:

        {% if greeting == 1 %}
            
Olá

        {% else %}
            
Tchau

        {% endif %}

Exemplo: for loop

templates/template.html:

        
    
            {% for x in mymembers %}
                
  • {{ x.firstname }}
    • 
            {% endfor %}

Referência de Tags

Uma lista de todas as tags de template:

Tag Descrição
autoescape Especifica se o modo de autoescape está ligado ou desligado
block Especifica uma seção de bloco
comment Especifica uma seção de comentário
csrf_token Protege formulários contra Cross Site Request Forgeries
cycle Especifica o conteúdo a ser usado em cada ciclo de um loop
debug Especifica informações de depuração
extends Especifica um template pai
filter Filtra o conteúdo antes de retorná-lo
firstof Retorna a primeira variável não vazia
for Especifica um loop for
if Especifica uma declaração if
ifchanged Usado em loops for. Exibe um bloco apenas se um valor foi alterado desde a última iteração
include Especifica o conteúdo/template incluído
load Carrega tags de template de outra biblioteca
lorem Exibe texto aleatório
now Exibe a data/hora atual
regroup Ordena um objeto por um grupo
resetcycle Usado em ciclos. Reinicia o ciclo
spaceless Remove espaços em branco entre tags HTML
templatetag Exibe uma tag de template especificada
url Retorna a parte URL absoluta de uma URL
verbatim Especifica conteúdos que não devem ser renderizados pelo mecanismo de template
widthratio Calcula um valor de largura com base na proporção entre um valor dado e um valor máximo
with Especifica uma variável a ser usada no bloco
