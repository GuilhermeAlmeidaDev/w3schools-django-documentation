Tag for do Django

Laços For

Um laço for é usado para iterar sobre uma sequência, como percorrer itens em um array, uma lista ou um dicionário.

Exemplo: Percorrer os itens de uma lista

    {% for x in fruits %}
  • {{ x }}
    • {% endfor %}

Exemplo: Percorrer uma lista de dicionários

    {% for x in cars %}

  • {{ x.brand }}

    {{ x.model }}

    {{ x.year }}

    • {% endfor %}

Dados de um Modelo

Dados em um modelo são como uma tabela com linhas e colunas.

{% for x in members %} {% endfor %}
id nome sobrenome telefone data_de_ingresso
{{ x.id }} {{ x.firstname }} {{ x.lastname }} {{ x.phone }} {{ x.joined_date }}

Exemplo: Percorrer itens buscados de um banco de dados

    {% for x in members %}

  • {{ x.id }}

    {{ x.firstname }} {{ x.lastname }}

    • {% endfor %}

Reverso

A palavra-chave reversed é usada quando você deseja fazer o loop em ordem reversa.

    {% for x in members reversed %}

  • {{ x.id }}

    {{ x.firstname }} {{ x.lastname }}

    • {% endfor %}

Vazio

A palavra-chave empty pode ser usada se você deseja fazer algo especial se o objeto estiver vazio.

    {% for x in emptytestobject %}
  • {{ x.firstname }}
    • {% empty %}
  • Sem membros
    • {% endfor %}

Variáveis de Loop

O Django possui algumas variáveis que estão disponíveis para você dentro de um loop:

    {% for x in fruits %}
  • forloop.counter: {{ forloop.counter }}
  • forloop.counter0: {{ forloop.counter0 }}
  • forloop.first: {{ forloop.first }}
  • forloop.last: {{ forloop.last }}
  • forloop.revcounter: {{ forloop.revcounter }}
  • forloop.revcounter0: {{ forloop.revcounter0 }}
    • {% endfor %}
