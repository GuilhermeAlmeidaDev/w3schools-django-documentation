Tag de Comentário do Django

Comentários permitem que você tenha seções de código que devem ser ignoradas.

Exemplo

Obtenha seu próprio servidor Django

Bem-vindos a todos

{% comment %}

Bem-vindas senhoras e senhores

{% endcomment %}

Descrição do Comentário

Você pode adicionar uma mensagem ao seu comentário, para ajudá-lo a lembrar por que você escreveu o comentário, ou como mensagem para outras pessoas que estão lendo o código.

Exemplo

Adicione uma descrição ao seu comentário:

Bem-vindos a todos

{% comment "esta era a mensagem de boas-vindas original" %}

Bem-vindas senhoras e senhores

{% endcomment %}

Comentários Menores

Você também pode usar as tags {# ... #} ao comentar o código, o que pode ser mais fácil para comentários menores:

Exemplo

Comente a palavra "Todos":

Bem-vindos{# Todos#}

Comentários em Views

As views são escritas em Python, e os comentários em Python são escritos com o caractere #:

Exemplo

Comente uma seção na view:


        from django.http import HttpResponse

        def testing(request):
            template = loader.get_template('template.html')
