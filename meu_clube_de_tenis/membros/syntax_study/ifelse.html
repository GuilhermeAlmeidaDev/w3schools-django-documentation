Declaração If

Uma declaração if avalia uma variável e executa um bloco de código se o valor for verdadeiro.

Exemplo:

        
        {% if cumprimento == 1 %}
            
Olá

        {% endif %}

Elif

A palavra-chave elif significa "se as condições anteriores não forem verdadeiras, então tente esta condição".

Exemplo:

        
        {% if cumprimento == 1 %}
            
Olá

        {% elif cumprimento == 2 %}
            
Bem-vindo

        {% endif %}

Else

A palavra-chave else captura qualquer coisa que não seja capturada pelas condições anteriores.

Exemplo:

        
        {% if cumprimento == 1 %}
            
Olá

        {% elif cumprimento == 2 %}
            
Bem-vindo

        {% else %}
            
Adeus

        {% endif %}

Operadores

Os exemplos acima usam vários operadores para comparar valores.

Exemplo:

        
        {% if cumprimento %}
            
Olá

        {% endif %}
        
    
        
        {% if cumprimento == 2 %}
            
Olá

        {% endif %}
        
    
        
        {% if cumprimento != 1 %}
            
Olá

        {% endif %}
        
    
        
        {% if cumprimento < 3 %}
            
Olá

        {% endif %}
        
    
        
        {% if cumprimento <= 3 %}
            
Olá

        {% endif %}
        
    
        
        {% if cumprimento > 1 %}
            
Olá

        {% endif %}
        
    
        
        {% if cumprimento >= 1 %}
            
Olá

        {% endif %}
        
    
        
        {% if cumprimento == 1 and dia == "Sexta-feira" %}
            
Olá fim de semana!

        {% endif %}
        
    
        
        {% if cumprimento == 1 or cumprimento == 5 %}
            
Olá

        {% endif %}
        
    
        
        {% if cumprimento == 1 and dia == "Sexta-feira" or cumprimento == 5 %}
            
Olá fim de semana!

        {% endif %}
        
    
        
        {% if 'Banana' in frutas %}
            
Olá

        {% else %}
            
Adeus

        {% endif %}
        
    
        
        {% if 'Banana' not in frutas %}
            
Olá

        {% else %}
            
Adeus

        {% endif %}
        
    
        
        {% if x is y %}
            
SIM

        {% else %}
            
NÃO

        {% endif %}
        
    
        
        {% if x == y %}
            
SIM

        {% else %}
            
NÃO

        {% endif %}
        
    
        
        {% with var1=x var2=x %}
            {% if var1 is var2 %}
                
SIM

            {% else %}
                
NÃO

            {% endif %}
        {% endwith %}
        
    
        
        {% if x is not y %}
            
SIM

        {% else %}
            
NÃO

        {% endif %}
